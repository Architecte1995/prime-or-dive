# Prime or Dive v6

A factorization clicking game with prime-number death mechanic.

## Gameplay

- Numbers fall from the top of the screen
- Click a composite number to split it into two factors
- **If you click a prime number, you die instantly**
- If a composite number hits the floor, you lose its value from your Pool
- Goal: reduce the initial Pool to zero by splitting every composite

## Tech Stack

- Pure HTML5 / CSS3 / Vanilla JavaScript
- No external libraries, no Canvas, no WebGL
- Web Animations API for particle effects
- Mobile-friendly (touch + click)

## Sacred Constants

- `MIN_VALUE = 37` — the 12th prime number
- `MAX_VALUE = 77773337` — a stretched prime-family variant of 73
